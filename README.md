# D Note

Simple, Clean & Light Weight Notes app.

The App is built with ~~XML~~ [Jetpack-Compose](https://developer.android.com/jetpack/compose)

## Features:
 - View All Notes
 - Adding a Note.
 - Updateing a Note.
 - Swipe to Delete a Note.
 - Undo Deletion.
 - Search Notes. 
 - Material You & Dark Mode Support!
